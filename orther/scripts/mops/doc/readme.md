# 关于日志定时备份传输说明
 程序分客户端**magent.py**和服务端**mserver.py**，通过url携带参数通讯。
 
服务端功能：
> - [x] 定时通过POST 推送开始压缩命令  
> - [] 收集压缩状态信息
> - [] 汇总压缩信息，发送邮件通知

客户端功能：
> - [x] 依照正则表达式压缩文件
> - [x] 删除压缩后的文件
> - [x] 推送压缩文件至备份服务器
> - [x] 清理本地过期压缩文件
> - [] 返回本地执行结果
